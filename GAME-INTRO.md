# 万物涨落 · 理想人生

> 浏览器单机人生模拟 · 30 年周制 · 职业 × 股市 × 关系 × 理想  
> 当前构建版本：`2026.06.09-extensions-batch2`（更新后请 **Ctrl+F5** 强刷）

## 一、游戏是什么

你在 **1988 年出生**，经历高考与离校，在 **约 30 年（1,560 周）** 里安排每周日程、求职、理财、经营关系、追逐理想。行业前景、裁员、股价、流量等宏观变化由 **「涨落」** 驱动——机会与风险同涨同落。

**通关目标（人生目标达成）**：还清房贷、自有住房、已婚且未离婚、非流浪。达成后可继续玩到 30 年自然结算。

**数据规模**：242 种职业 · 10,000 家企业（100 大型 / 900 中型 / 9,000 小型）· 24 只 A 股真实标的（每档随机抽取）。

## 三、创造人生（新开局）

选档后点击 **🎲 创造人生**，通过骰子与轮盘决定出身与命运（不可逐步撤销，请慎重押注）。

### 3.1 七步流程

| 步骤 | 问题 | 押注规则 | 结果 |
|------|------|----------|------|
| 1 | 你的性别 | 男（蓝筹码）/ 女（粉）/ 不知道（随机并告知） | 决定 `playerGender` |
| 2 | 喜欢的性别 | **大**=男人 · **小**=女人（仅此一枚筹码） | 玩家性向固定为 **双性恋**；押中则偏好明确，押错则仍偏好所选性别，也会欣赏另一性别 |
| 3 | 家庭贫富 | **大**=富有 · **小**=朴素 | 富有 / 正常 / 贫穷；押大家庭多零花钱但亲子熟悉度偏低（约 42），押小相反（约 78） |
| 4 | 实力或压力 | **大**=实力（高三维）· **小**=低压起点 | 原始压力 0～100；影响开局三维模式 |
| 5 | 高考命运 | 投 3 次骰子，每次最高 216，合计最高 648 | 至少 2 次押中「大」可 **再投 1 次**（最高 864）；3 次全大→心智偏高；0 次大→肉体偏高 |
| 6 | 有没有梦想 | **大**=有 · **小**=不知道 | 影响学历、专业与理想项目 |
| 7 | 轮盘赌 | 红=生命 · 黑=金钱 · 或押数字 0～36 | 见下表 |

### 3.2 轮盘结果（第一桶金）

> 「第一桶金很重要，当然活着才有命花钱。」

| 押注 | 开出 | 效果 |
|------|------|------|
| 红 | 红 | 预期寿命 + 开奖数字 |
| 红 | 黑 | 预期寿命 − 开奖数字（不低于 40） |
| 黑 | 黑 | 祖辈遗产 = 数字 × ¥10 万（需等父母离世领取） |
| 黑 | 红 | 现金 + 数字 × ¥10 万，寿命折损 |
| 数字 **0** | 命中 | **毕业月**遇见命中注定之人（你还不知道是谁） |
| 数字（**红**） | 命中 | **毕业后第 N 年**遇见命中注定感情（N = 开奖数字，身份未知） |
| 数字（**黑**） | 命中 | **毕业后第 N 年**彩票中奖（N = 开奖数字） |

### 3.3 高考 → 学历与院校

| 分数 | 学历 | 院校 |
|------|------|------|
| < 280 | 高中/中专 | — |
| 280～379 | 大专 | 普通高校 |
| ≥ 380 | 本科 | 普通高校 |
| ≥ 480 | 本科 | 985/211 |
| ≥ 580 | 本科 | C9/常春藤 |
| 特殊（有梦想 + 高分） | 博士 | 按分数 |
| 特殊（梦想押大但骰子小） | 硕士 | 需 ≥ 420 分 |

### 3.4 开局背景

- **1988 年生**，**2010 年离校还有一个月**，寝室即将到期，年龄 **22 岁**。
- 姓名 **随父母姓氏自动生成**（名随机）；父母为通讯录中独立的 **父亲 / 母亲** 联系人。
- 旧档续玩若尚未正式取名，会弹出：**「你终于想重新开始用自己多年的姓名…」** 并显示父母姓名。
- 默认现金：人生创造约 **¥5,000 起**（轮盘可改）；非创造流程默认 **¥50,000**。
- 开局自动入职 → **满月（4 周）裁员剧情**（校招后第一次职业震荡）。
- 社交圈自动建立：**小学 / 初中 / 高中 / 大学同学圈**。

## 五、属性、压力与寿命

### 5.1 三维

| 属性 | 说明 | 上限 |
|------|------|------|
| 肉体 | 体力、健康相关 | 120 |
| 心智 | 学习、脑力 | 120 |
| 精神 | 情绪、意志 | 120 |

创建时三维总和约 **240**（创造人生按「实力/压力」步决定高低或随机）。  
**有效属性** = 基础三维 + 临时加成（±10）+ 女性生理周期修正（±10）。

### 5.2 压力

| 阈值 / 事件 | 效果 |
|-------------|------|
| 原始压力 `baseStress` | 创造人生决定；长期有向心力回归（设计方向） |
| 失业 | 每周 +1 |
| 单身 | 每周 +5 |
| 异地 | 每周 +1 |
| 成功入职 | −30 |
| 孩子出生 | +50 |
| 离婚 | +25 |
| 压力 ≥ 666 | 无法胜任脑力劳动 |
| 压力 ≥ 1000 | 精神崩溃相关终局 |
| 压力 ≥ 60 连续 4 周 | 触发「海绵宝宝」勋章；并关联 **减寿** 设计（连续 4 周 >100 警告后减 1 年预期寿命） |

**存款里程碑减压**（首次达到）：50 万 / 100 万 / 500 万 / 1000 万 / 5000 万 / **1 亿** 各 −20 压力。

### 5.3 寿命与自然死亡

创造人生轮盘设定基础 **80 ± 5 岁**；红黑押注可增减。**玩家、父母、伴侣、通讯录中的每个人** 都有独立的 **预期寿命** 与隐藏的 **死亡周**（具体时刻未知，在命数附近随机）。

| 规则 | 说明 |
|------|------|
| **自然死亡** | 到达预定死亡周时 **寿终**；这是人物离世的 **唯一常规途径** |
| **突发死亡** | 与寿命 **并行、独立**；目前仅有 **通宵猝死**（连续第 5 天通宵不睡） |
| **后续扩展** | 事故、疾病等突发死亡规则将另行加入，**不替代** 寿命系统 |

**父母离世与遗产**：

- 父母按各自寿命离世；**双亲均离世后** 才结算遗产（含轮盘「黑/black」的 **祖辈遗产**）。
- **不再** 存在「啃老满 10 年 → 父母死亡」的旧规则。
- 收入不足时，只要 **至少一位父母在世**，未婚或离异状态下仍可 **啃老**（无时间上限）。

**伴侣 / 联系人**：到达寿命同样会离世；通讯录显示「已离世」，无法再联系。

## 六、通讯录与人际

### 6.1 核心联系人

| 角色 | 说明 |
|------|------|
| 父亲 / 母亲 | 独立联系人，有姓名、职业、熟悉度；可电话（白天/晚上） |
| 配偶 | 婚姻对象，亲密度系统 |
| 基友/闺蜜 | 可约外出（公园、咖啡、图书馆、夜店、酒吧、便利店） |
| 前配偶 | 离婚后保留；涉及抚养、亲子鉴定等 |

### 6.2 熟悉度与关系等级

| 熟悉度 | 关系 |
|--------|------|
| < 60 | 陌生人 |
| ≥ 60 | 朋友 |
| ≥ 80 | 好友（可邀请别墅影院、派对等） |
| ≥ 90 | 挚友（可帮 TA **建立理想**） |

**吸引力**（0～100）：与性向共同决定是否可发展亲密关系。

| 吸引力 | 效果 |
|--------|------|
| ≥ 70 且双方性向匹配 | 可发展 **亲密关系**，**不必**先刷满熟悉度 |
| ≥ 75 | 对方可能 **主动表白**（每周约 12% 概率） |
| 表白被拒 | 熟悉度下降；吸引力越高掉得越多 |

### 6.3 性向

| 对象 | 规则 |
|------|------|
| 玩家 | 开局固定 **双性恋** |
| 配偶关系 | 同性=同性恋情侣 · 异性=异性恋情侣 |
| 联系人 | **同性恋 / 异性恋 / 未知**（随机） |

### 6.4 亲密关系（原「出轨」）

- 通讯录标签改为 **「亲密关系」**，不再使用「出轨」字样。
- 满足条件后可 **幽会**（酒店 ¥300 / 五星 ¥2000 / 户外等）。
- 已婚时触发 **婚外情状态**，**月生活费 ×2**。
- 可发展为炮友、求婚、秘密成婚；含怀孕要挟、限期婚礼、告密、勒索等剧情。
- **监禁**：严重冲突或强奸指控约 **15 天（2 周）**，期间无法日常、求职、幽会。

### 6.5 其他社交

- 每时段每人可联系 **1 次**；**22%** 无人接听。
- 熟人可 **内推**、**借钱**（借出后按期还款或赖账）。
- 结识新人：外出、派对、职场故事等；显示 **结识 · 日期 · 地点**。
- 幽会、结识、段子等走 **person 弹窗队列**，避免与神器弹窗冲突。

## 七、网络与圈子

### 7.1 网络页签

- 需 **手机或电脑** 才能使用。
- 通讯录只显示 **概况**；点 **🌐 详情** 跳转网络查看完整档案。
- 可查看 **玩家本人** 与所有联系人的详情。

### 7.2 个人详情包含

- 姓名、性别、年龄、出生城市、三维、性向、公众 **影响力**
- **职业**：喜好职业、主职经验（% 与 实习生/普通岗/专家/总监）、从业公司与城市
- **学历经历**（按时间排列）
- **理想**：项目描述 + 目标职业；进行中显示「理想实现中」
- **圈子**：社交圈 / 爱好圈 / 职场圈

### 7.3 三类圈子

| 类型 | 构成 | 说明 |
|------|------|------|
| **社交圈** | 熟悉度 + 吸引力 | 玩家开局有小学～大学四圈；成员间有小社交关系 |
| **爱好圈** | 共同喜好与理想 | 可开展小项目（设计：按周期结算 流产/失败/成功，+1～10 职业经验） |
| **职场圈** | 当前工作关系 | 组员圈、职能层级、部门、全公司人脉；离职后熟人转入社交圈 |

**外出带伙伴**（设计目标）：日常外出可选 **1～3 位** 伙伴，共同形成/强化圈子。

**别墅管家、公司保镖司机**（设计目标）：作为具体 NPC 归入职场圈。

## 八、梦想与理想项目

### 8.1 个人理想

每人可有 **理想项目**，描述如何完成，通常对应 **理想职业**，并需多种职业协作支持。

**模板示例**：

| 理想 | 目标职业 | 协作需求 |
|------|----------|----------|
| 独立电影 | 演员 | 导演、摄影、制片… |
| 个人专辑 | 歌手 | 制作人、编曲、宣传… |
| 独立游戏 | 游戏制作人 | 程序、美术、策划… |
| 地标方案 | 建筑师 | 结构、项目、造价… |
| 专利产品 | 工程师 | 研发、测试、市场… |
| 长篇作品 | 作家 | 编辑、插画、发行… |

### 8.2 职业经验

| 经验值 | 档位 |
|--------|------|
| < 30% | 实习生 |
| 30%～69% | 普通岗 |
| 70%～99% | 专家 |
| ≥ 100% | 总监 |

- 爱好圈项目：成功/失败均可 **+1～10** 经验；**流产**（未按时参加）无经验。
- 联系人概况通常显示 **最高的一项** 职业经验。
- 玩家可 **赞助** 他人爱好（设计：每项爱好有一笔投资需求）。

### 8.3 公司侧理想

注册公司后需选 **行业**、建 **品牌**（可与公司同名）；品牌与产品有 **熟悉度 / 影响力**，**更名重置**。

经营 **服务 / 制造** 类项目（部分需研发），有周期与组织架构。玩家通过公司项目 **帮助任何人完成理想**（打工运营 + 理想构筑的核心玩法方向）。

## 九、日常生活

### 9.1 时间结构

- **7 天一周**；每天 **白天 / 晚上 / 后半夜** 三个时段。
- 每时段 **8 小时** 行动预算；部分活动占满时段或跨时段。
- 本月 **旷工上限 4 次**（玩家与伴侣一致）。

### 9.2 宅家

- 刷手机、短视频、手游、零食、_console_、电脑等。
- 短视频：每周最多 **60** 次点击；垃圾内容 + 压力，有趣内容减压。
- 别墅活动（需先购别墅，见第十章）。

### 9.3 外出

- 公园、图书馆、咖啡、酒吧、夜店、商场等。
- 消耗时段；可能 **结识** 或 **再遇** 熟人；触发 **职场八卦**（按地点不同概率）。
- 与基友/闺蜜 **履约赴约**，爽约 + 压力。

### 9.4 消费与装备（部分）

| 物品 | 价格 |
|------|------|
| 游戏机 | ¥3,000 |
| 电脑 | ¥10,000 |
| 小米 / 诺基亚 / iPhone / 华为 | ¥2,000 / ¥399 / ¥10,000 / ¥18,000 |
| 代步车 / SUV / 跑车 | ¥15 万 / ¥30 万 / ¥90 万 |
| 约会 | ¥500 |
| 单人餐 / 双人餐 | ¥50 / ¥100 |

### 9.5 通宵与风险

- 连续通宵有 **猝死** 风险：第 5 天 **100%**；随年龄递增。
- 后半夜时段耗尽需选择 **睡觉** 或继续通宵。

## 十、房产、别墅与公司

### 10.1 住房

| 项目 | 数值 |
|------|------|
| 开局 | 自有房 + 房贷 |
| 月供 | ¥6,000 × **360 期** |
| 一次性还清 | **¥300 万** |
| 离异后 | 保留房则月供升至 ¥12,000；或租房 ¥3,000/月 |

**租房**（设计）：群租 ¥500 / 合租 ¥1000 / 独租 ¥2000；亦可住家里。

### 10.2 别墅 · ¥1,000 万

| 设施 | 建造费 | 活动 | 效果 |
|------|--------|------|------|
| 私人图书馆 | ¥200 万 | 研究 **8h**（占满时段） | 心智 +1 · 压力 −1 |
| 静室 | ¥200 万 | 冥想 **1h** | 压力 −10 |
| 私人影院 | ¥200 万 | 看电影 **2h** | 压力 −1；可邀 **1 位好友**，熟悉度 +1（配偶则亲密度 +1） |
| 管家 | — | 月费 **¥10 万** | — |

**别墅派对**（每日 1 次，可邀好友，不足人数由陌生人补位）：

| 规模 | 人数上限 | 费用 | 减压 |
|------|----------|------|------|
| 小型 | 10 | ¥10 万 | −8 |
| 中型 | 50 | ¥20 万 | −10 |
| 大型 | 100 | ¥50 万 | −12 |
| 超级 | 200 | ¥100 万 | −15 |

派对认识新人走 **逐人弹窗**；邀请好友 **熟悉度 +2**。

### 10.3 注册公司 · ¥5,000 万

- 月费团队（保镖司机等）**¥20 万/月**。
- 开公司时需 **为公司命名**；品牌熟悉度 / 影响力初始约 **5 / 3**，**更名重置**。
- 旧档已开但未命名会 **强制补名**。
- **收购** 本人或伴侣的就职企业：估价后现金收购，之后 **可不去上班且不会被辞退**；持股绑定真实股票（按企业名精确匹配，如宁德时代 → 300750）。
- **出售公司**：约注册价 35% + 持股市值。
- **国有股份企业** 不可收购（设计规则）。

### 10.4 公司管理（老板玩法）

| 功能 | 说明 |
|------|------|
| 📬 招聘邮箱 | 发布岗位 → 每周收简历 → HR / 主管建议 → 玩家一票录取或否决 |
| 🏢 组织图 | 总裁办、人力、财务、业务部、技术部 |
| 📊 周报 | 员工贡献、项目进度、办公室八卦 |
| 📈 月报 | 项目盈亏、运营开支、持股波动；可 **新开行业**（¥800 万）或 **新立项** |
| 立项编制 | 精干小队 / 标准编制 / 重投入 |
| 项目 | 含 **品牌名 + 产品名**（可选）；各有熟悉度 / 影响力，更名重置 |

**公司架构（设计方向）**：董事会股权、CEO/商务/人事/技术高管、中层管行业大项目、基层管具体周期项目；改选需投票，超级投票权可一票否决。

## 十一、求职与职场

### 11.1 渠道

| 渠道 | 费用 | 特点 |
|------|------|------|
| 线下人才市场 | ¥200/场 | 8 小时，8 摊位，最多投 48 家 |
| 招聘 APP | ¥100/平台/月 | BOSS、拉勾、猎聘、智联、前程 |
| 猎头 | 入职收年薪 **20%** | 可欠账从工资扣 |
| 内推 | 免费 | 本周约 3%～7% 概率出现 |

### 11.2 应聘流程

选职业 → 选渠道 → 投递 → 收回复（Inbox）→ 约面试 → 到场 → Offer → 谈薪签约 → 入职

- 面试交通：同城 ¥50 起；远城含住宿更高。
- 部分行业有 **年龄上限**（如 IT 40 岁、金融 42 岁）。
- 学历与院校影响通过率；C9/985 擅长大/中型企业。

### 11.3 职场生活

- 上班占时段；可 **加班**、 **摸鱼**、 **请假**。
- 公司 **裁员** 与行业 AI 热度相关； sponge 勋章可见裁员率。
- **满月裁员** 开局剧情；之后随宏观与行业涨落波动。
- **职场八卦**：外出时熟人可能讲行业/公司故事。

### 11.4 诈骗岗（设计）

- 部分招聘为 **诈骗**：投必中，配 **特殊通讯簿**，工作即给名单打电话（与后续项目运营玩法衔接）。

## 十三、经济系统

### 13.1 炒股

- **24 只** A 股，锚点价约 2024 年初；**不占本周行动**。
- 佣金 **万分之 2**。
- 个人 `portfolio` 与公司 `holdings` **分开**；收购持股至少约 **1 万股**，勿与炒股默认 **100 股** 混淆。

### 13.2 赌博（澳门五日游）

- 骰宝、轮盘等；独狼勋章：**12%** 败局翻盘。
- 阶段等待时间已减半（开盅动画不变）。

## 十四、家庭、伴侣与生育

### 14.1 婚姻

- 开局 **已婚**（命中注定爱情轮盘可改相遇节奏）；亲密度初始 **80**（范围 −100～200）。
- **约会** ¥500 · 压力 −5 · 亲密度 +1；**双人餐** 亲密度 +1。
- 每周性爱上限 **7** 次；**线上约会** 可异地维持关系。

### 14.2 伴侣平行人生

- 伴侣独立 **求职、上班、炒股、内推**；状态可在 **家庭** 页查看。
- **零花钱** ¥50～1,000/周；**秘密小金库** 离婚或揭露后才可见。
- 伴侣 **每月旷工 4 次** 与玩家一致；叫回家、跟出去玩、出轨风险等完整互动。

### 14.3 生育

| 方式 | 费用 | 说明 |
|------|------|------|
| 自然怀孕 | 避孕 6% / 不避孕 20% / 备孕 35% | 孕期 40 周 |
| 备孕 | ¥3,000 | 提高下次怀孕率 |
| 国内/国外代孕 | ¥50 万 / ¥100 万 | 交货约 12 月 |
| 国内/国外试管 | ¥10 万 / ¥20 万 | 可借精 |
| 堕胎 | ¥3,000 | 仅限孕 **≤12 周** |
| 亲子鉴定 | ¥5,000 | 抚养纠纷等 |

**女性生理周期**（21～35 岁）：经期 5 天；排卵窗 ±2 天；属性 ±10 波动；孕早期同房有流产风险。

### 14.4 离婚与挽回

- 律师费按婚内资产计算；过错方影响分割。
- 玩家主动离婚：保留房产，房贷升高。
- **愿望值 ≥ 50** 可和解；和解后亲密度 **120**，通讯录清除「前妻/前夫」残留。

## 十八、终局与失败

| 终局 | 条件 |
|------|------|
| 人生目标达成 | 还清房贷 + 有房 + 已婚未离 + 非流浪 |
| 30 年到期 | 正常结算 |
| 命数已尽 | 到达预期寿命（自然死亡） |
| 通宵猝死 | 连续第 5 天通宵（**突发死亡**，与寿命无关） |
| 伤残 + 无房 | 「出租屋里的沉默」 |
| 流浪 24 月 | 「桥洞底的寒夜」 |
| 精神崩溃 | 压力极高持续 |

**生存链**：收入不足 → **啃老**（父母在世即可，无 10 年上限）→ 父母离世后 **遗产** → 无力维持 **低保** → 伤残低保 → **流浪**。
